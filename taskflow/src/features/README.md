# Feature Modules

Feature-specific code should live here as the refactor progresses.

- `auth`: authentication screens, hooks, and auth-specific actions
- `projects`: project pages, project settings, and membership UI
- `tasks`: kanban, task detail, task chat, and task actions
- `messages`: direct/group messaging UI and hooks
- `meetings`: meeting room and meeting dashboard code
- `admin`: user management and admin workflows
